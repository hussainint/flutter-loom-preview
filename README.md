 
Preview Loom.com Video in Flutter Android, IOS and Web

## Features

Display a Loom Video and redirect on tap


## Usage

```dart
LoomVideoPreview(
        loomLink: 'https://www.loom.com/share/8ff11b1edd6b490b93af7d2eb6f91862',
        loadingWidget: CircularProgressIndicator(),
        previewHeight: 250,
        previewWidth: 250,
      )
```
